# Custom CSS

## Set up

Follow the guide at [Portkey Sign In](/portkey-sign-in).

## Open the Inspector

Open the inspector in Chrome by pressing the `F12` key on your keyboard, or just right-click on the page and select the Inspect option.

Now, click on the Sign In button.

The Login modal appears:

![Portkey Sign In Modal](/portkey-sign-in-modal.png)

Right-click on the blue button "Login with Phone/Email" and select `Inspect`.

In the Inspector, notice that the style for this button is as follows:

```css
.portkey-ui-base-modal .portkey-ant-btn-primary,
.portkey-ui-common-modals .portkey-ant-btn-primary,
.portkey-ui-wrapper .portkey-ant-btn-primary {
  background-color: var(--portkey-ui-bg-2);
  border-color: var(--portkey-ui-border-3);
}
```

## Change the background colour of the sign in button

Now, let's try to change the background colour.

Create a new file `src/App.css`:

```css filename="src/App.css" copy
:root {
  --portkey-ui-bg-2: #129365;
}
```

Now, let's import this file:

```js filename="src/App.js" copy
import "./App.css"; // import your custom css

// ... rest of your code
```

Notice that the colour of the button has changed:

![Portkey Sign In Modal](/portkey-sign-in-modal-green.png)

Congratulations! You have successfully customised the Sign In component's button.

## Further customisation

For more advanced customisations, you may inspect and target specific css classes, but do note that you should test out your customisations to ensure that nothing looks weird.
