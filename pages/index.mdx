# Introduction

This is a demo site created for the Portkey Developer workshop.

# Objective

The objective of this workshop is to get familiar with how to:

- [Create a new token](/create-token) on AElf Chain using `aelf-command`.
- [Deploy a sample contract](/deploy-smart-contract) to AElf node.
- Use `@portkey/did-ui-react` to [authenticate](/portkey-sign-in) with Portkey Wallet, call the [getBalance](/get-balance) method, and [get NFT](/get-nft).
