# Introduction

This is a demo site created for the Portkey Developer workshop.

# Objective

The objective of this workshop is to get familiar with how to:

- Use `@portkey/did-ui-react` to [authenticate](/portkey-sign-in) with Portkey Wallet and to call the [getBalance](/get-balance) method.
- [Create a new token](/create-token) on AElf Chain using `aelf-command`.
- [Deploy a sample contract](/aelf-developer-tools) to AElf node.
